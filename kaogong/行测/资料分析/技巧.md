## 1.2  减法技巧--整数基准法、"21" "12" 分段法

### 1.2.1 整数基准法

减数减一减数 = (减数减一基准值) + (基准值减一减数)；

例如 "632-589"，我们可以加入 600 作为基准值，则 632-589 = (632-600) + (600-589)。

### 1.2.2 "21" "12" 分段法

将三位数的减法分成 "21" 或 "12" 两段，尽可能保证不用借位。

## 1.3 乘法技巧一-小分数、拆分法

### 1.3.1 小分数

若乘法中某个乘数可以以简单的化分数形式出现，我们就可以将多位数乘法转化为简单算法计算。

**需记忆的常用小数如下：**

- 50% = 1/2
- 33.3% = 1/3
- 25% = 1/4
- 20% = 1/5
- 16.7% = 1/6
- 14.3% = 1/7
- 28.6% = 2/7
- 42.9% = 3/7
- 12.5% = 1/8
- 11.1% = 1/9

**[例题]**

- 784 x 25%
- 632 x 12.5%
- 727 x 16.7%
- 981 x 33.3%
- 20% x 455
- 50% x 472

## 1.3.2 乘法拆分

若乘法中某个乘数为百分数且能拆成两个简单数值（50%、10%、5% 等），我们可以将该百分数拆成两个部分，要遵循“1%（“一个包干”）”。

**常用的拆分方法如下：**

- 45% = 50% - 5%
- 55% = 50% + 5%
- 15% = 10% + 5%
- 60% = 50% + 10%
- 95% = 1 - 5%
- 90% = 1 - 10%

以及所有 50%、100%  附近的数字等等。

## [例题]

- 382 x 55%
- 492 x 48%
- 936 x 45%
- 592 x 97%
- 1228 x 95%
- 766 x 103%
